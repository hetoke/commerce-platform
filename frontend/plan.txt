Web e-commerce shop app?

Feature:

- CRUD (merchandises)
- Multiusers (sellers, buyers) -> Login / Logout (JWT)

- Buy (buyers): Select to an item -> Buy buttons


Data definitions:
Item [name, price, location of sellers, description]
Users [username, password, role]

{
	username,
	password,
	role,
	items: [
		{
			name,
			price,
			location,
			description
		},
		{
			name,
			price,
			location,
			description
		}
	]

}



Tech stack
Frontend: ReactJS, TailwindCSS, Vite
Backend: NodeJS + ExpressJS
Database: MongoDB

Arch: MVC
